## Install

<Tabs>
  <Tab title="macOS">
    1. Open the downloaded `.dmg` file
    2. Drag **Brunel Agent** into your **Applications** folder
    3. Open Brunel Agent from Applications or Spotlight

    The first time you open the app, macOS may display a security prompt. Follow the on-screen instructions to allow it to run.
  </Tab>
  <Tab title="Windows">
    1. Run the downloaded `.exe` installer
    2. Follow the installation prompts
    3. Launch Brunel Agent from the Start menu or desktop shortcut
  </Tab>
</Tabs>

## Auto-Updates

Brunel Agent checks for updates automatically each time it launches. Updates come in two modes:

| Mode | Behavior |
|------|----------|
| **Optional** | A notification banner appears at the top of the app. Click **Update Now** to install, or dismiss it and update later. |
| **Mandatory** | A full-screen prompt blocks the app until the update is installed. This is used for critical releases that require all users to be on the same version. |

In both cases the update downloads in the background. When it is ready you'll see the option to restart and apply it immediately.

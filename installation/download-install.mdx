## System Requirements

| Platform | Requirements |
|----------|-------------|
| **macOS — Apple Silicon** | M1, M2, M3, or M4 chip. macOS 12 (Monterey) or later. |
| **macOS — Intel** | Intel processor. macOS 12 (Monterey) or later. |
| **Windows** | Windows 10 or later (64-bit). |

## Download

Visit [download.brunelagent.ai](https://download.brunelagent.ai). The page auto-detects your OS and highlights the right installer. For other platforms, select from **Other Platforms**.

## Install

<Tabs>
  <Tab title="macOS">
    1. Open the downloaded `.dmg` file
    2. Drag **Brunel Agent** into your **Applications** folder
    3. Open Brunel Agent from Applications or Spotlight

    The first time you open the app, macOS may display a security prompt. Follow the on-screen instructions to allow it to run.
  </Tab>
  <Tab title="Windows">
    1. Run the downloaded `.exe` installer
    2. Follow the installation prompts
    3. Launch Brunel Agent from the Start menu or desktop shortcut
  </Tab>
</Tabs>

## Auto-Updates

Brunel Agent checks for updates automatically each time it launches. When a new version is available, it will download and install in the background. You'll be prompted to restart the app to apply the update.

---
title: "Keyboard Shortcuts"
description: "Keyboard shortcuts for Brunel Agent on macOS and Windows."
---

Brunel Agent supports keyboard shortcuts to speed up common actions. Shortcuts use `Cmd` on macOS and `Ctrl` on Windows.

---

## Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl` + `N` | Create a new session in the current project |
| `Cmd/Ctrl` + `1` – `9` | Switch to session tab 1–9 |

## Chat

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl` + `Enter` | Send message |
| `Escape` | Cancel current action or close dialog |

## Files

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl` + `Shift` + `F` | Toggle the Context Files panel open/closed |
| `Cmd/Ctrl` + `F` | Search within the file viewer (when a file is open) |
